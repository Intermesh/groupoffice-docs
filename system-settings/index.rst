System settings
===============

When you click on the avatar icon on the top right you can open the 
"System settings" page.

.. figure:: /_static/system-settings/general.png

   System settings

This is the place to configure everything for Group-Office. When setting up 
Group-Office we recommend to review each tab and make adjustments where 
necessary before you start creating users and start using the system.
